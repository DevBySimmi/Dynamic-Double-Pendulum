# Dynamic-Double-Pendulum
A visually enhanced double pendulum simulation using RK4 integration, motion trails, energy graphs, and real-time physics animation in Python.

---

## 🚀 Features

- 🎞 Smooth real-time animation
- 🌀 Motion trail visualization
- 🌌 Dark theme UI
- ⚡ RK4 numerical integration
- 📊 Energy vs Time graph
- 🎨 Enhanced pendulum visuals
- ⏱ Real-time simulation timer
- 💾 GIF export support

---

## 🧠 Physics Concepts Used

- Double Pendulum Dynamics
- Chaotic Motion
- RK4 (Runge-Kutta 4th Order)
- Conservation of Energy
- Numerical Simulation
